Descrição do Projeto

War of Israely é um jogo de estratégia desenvolvido em Python utilizando a biblioteca Pygame. Inspirado no clássico jogo WAR, o projeto desafia os jogadores a conquistar territórios, administrar exércitos e planejar ataques estratégicos para dominar o mapa mundial.

O jogo possui uma interface gráfica interativa, animações de batalha com dados, sistema de cartas de recompensa e gerenciamento de tropas, proporcionando uma experiência dinâmica e divertida para os jogadores.

Objetivo do Jogo

O objetivo é conquistar todos os territórios do mapa e eliminar os adversários. Para isso, os jogadores devem distribuir tropas, atacar territórios inimigos, fortalecer suas fronteiras e utilizar estratégias para expandir seu império.

Como Jogar

Cada jogador recebe territórios distribuídos aleatoriamente no início da partida. Durante seu turno, o jogador passa por três fases:

Distribuição

O jogador recebe tropas de reforço com base na quantidade de territórios que controla e deve distribuí-las entre seus territórios.

Ataque

O jogador pode atacar territórios inimigos conectados aos seus. As batalhas são decididas por lançamentos de dados, e o vencedor pode conquistar novos territórios.

Remanejamento

Após os ataques, é possível mover tropas entre territórios aliados conectados para fortalecer posições estratégicas.

Sistema de Cartas

Ao conquistar pelo menos um território durante o turno, o jogador pode receber uma carta. As cartas podem ser trocadas por tropas extras quando determinadas combinações são obtidas, oferecendo vantagens estratégicas ao longo da partida.

Principais Funcionalidades
Interface gráfica desenvolvida com Pygame;
Mapa mundial interativo;
Sistema de turnos para até 4 jogadores;
Distribuição e movimentação de tropas;
Sistema de batalhas com dados;
Animações visuais durante os combates;
Sistema de cartas e bônus de reforço;
Registro de eventos e histórico de batalhas.
